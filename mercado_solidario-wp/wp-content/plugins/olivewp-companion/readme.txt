==== Description ====

OliveWP Companion plugin enhances the functionality of OliveWP theme. This plugin requires OliveWP theme to be installed.

==== Installation ====

1. Upload olivewp-companion.zip to the /wp-content/plugins/ directory and extract.
2. Activate the plugin through the Plugins > Installed Plugins menu in WordPress dashboard.
3. Done!

==== Key Features ====

* Trending Post
	* General
	* Typography setting
	* Color setting
* Starter Sites 
	* Demo Import features
